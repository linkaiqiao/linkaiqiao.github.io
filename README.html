📋 项目简介

课程表管理系统是一款专为学生设计的Android课程管理应用。帮助学生高效管理课程安排，支持账号密码登录和微信快捷登录，提供可视化周视图课表、课程管理、提醒设置等功能。

✨ 功能特性

🔐

安全登录

支持账号密码登录和微信快捷登录，安全便捷

📅

可视化课表

直观的周视图课程表，不同课程显示不同颜色

➕

课程管理

轻松添加、编辑和删除课程，支持批量操作

🔔

课程提醒

自定义提醒时间，课前自动提醒

⏰

时间设置

自定义上课时间和课间休息时长

📱

学期管理

支持多个学期课表切换管理

📖 使用说明

注册账号

使用手机号注册账号，或直接使用微信快捷登录

创建课表

选择学期和年份，创建您的课程表

添加课程

输入课程名称、时间、地点等信息，添加课程

设置提醒

设置课前提醒时间，不错过任何课程
